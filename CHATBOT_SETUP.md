# مشروعك - المساعد الذكي (Chatbot)

هذا المشروع يحتوي على تطبيق المحادثة الذكية لشركة "مشروعك"، باستخدام الواجهة الأمامية (React) والخلفية (Node.js/Express) لضمان أمان الـ API Keys.

## المتطلبات الأساسية
- Node.js (إصدار 18 فما فوق)
- MongoDB (اختياري، يعمل بذاكرة مؤقتة in-memory إذا لم يتوفر)

## هيكل المشروع المستحدث
- `server/` - يحتوي على الخادم (Backend)
  - `server/index.ts` - التطبيق الرئيسي للخادم والـ API
  - `server/models/Conversation.ts` - نموذج قاعدة بيانات MongoDB لحفظ المحادثات (اختياري)
- `src/ChatBot.tsx` - مكون المحادثة (Frontend) في React
- `.env` - ملف متغيرات البيئة الذي يحتوي على الـ API Keys

## طريقة التشغيل محلياً

1. **تثبيت الاعتماديات:**
تأكد من تثبيت الحزم المطلوبة، تم إضافة `express`, `mongoose`, و `dotenv` للخادم:
```bash
npm install
```

2. **التأكد من ملف .env:**
يجب أن يحتوي ملف `.env` في جذر المشروع على المفاتيح التالية:
```env
# مفتاح API الخاص بالشات بوت (Gemini 2.5 Flash Lite)
CHATBOT_API_KEY="AIzaSyDt0JpJDVOWnIZsSiSApVEEc4Lm5luojjg"

# منفذ الخادم ورابط الواجهة الأمامية
APP_URL="http://localhost:3000"
CHAT_PORT=3001

# رابط قاعدة بيانات MongoDB (اختياري)
MONGODB_URI="mongodb://localhost:27017/mashrouak-chat"
```

3. **تشغيل المشروع:**
تم إضافة أمر جديد لتشغيل الواجهة الأمامية والخلفية في نفس الوقت:
```bash
npm run dev:all
```
أو يمكنك تشغيل الخادم والواجهة كل على حدة:
- لتشغيل الخادم: `npm run server`
- لتشغيل الواجهة الأمامية: `npm run dev`

## مميزات النظام
- **أمان عالي**: مفاتيح جيمناي آمنة في الخادم ولا تظهر للمستخدم.
- **تخزين المحادثات**: يدعم MongoDB للاحتفاظ بسجل المحادثات، مع دعم التخزين المؤقت في حال عدم توفر قواعد البيانات.
- **تحديد حجم البيانات**: يتم إرسال آخر 20 رسالة فقط للذكاء الاصطناعي لتجنب تجاوز حد الـ Tokens.
- **واجهة عصرية**: تصميم زجاجي أنيق متوافق مع الهوية البصرية، مع رسوم متحركة واستجابة للهاتف المحمول.
- **ردود سريعة**: خيارات سريعة مبرمجة مسبقاً (Quick Replies).

## جاهزية الإنتاج (Production)
للإنتاج الفعلي (Deployment):
1. قم برفع المجلد `server` إلى بيئة Node.js (مثل Render, Heroku) وتأكد من إعداد متغيرات البيئة שם.
2. قم ببناء الواجهة الأمامية `npm run build` وارفعها إلى استضافة الويب (مثل Vercel, Netlify).
3. في ملف `src/ChatBot.tsx`، قم بتحديث المتغير `CHAT_API` ليشير إلى رابط الخادم الفعلي الذي قمت برفعه للحصول على الخدمة بشكل كامل.
